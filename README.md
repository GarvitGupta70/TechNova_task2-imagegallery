# Image Gallery

## Description

A responsive image gallery built using HTML, CSS, and JavaScript. It allows users to view images, navigate between them, and filter images by category.

## Features

* Responsive image gallery
* Next and Previous image navigation
* Hover effects and smooth transitions
* Mobile-friendly design
* Image category filters

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your web browser.
